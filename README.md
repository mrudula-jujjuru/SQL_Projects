### Project : Student Course Completion Prediction
“This project showcases my ability to leverage cutting-edge AI tools to accelerate development while maintaining a deep technical understanding of the underlying data architecture.”
1. Project Methodology
This project utilized an iterative AI-assisted workflow to ensure logical accuracy and adherence to relational database standards.This process served as a practical learning vehicle for Views,StoredProcedures, allowing me to validate AI suggestions against industry best practices.
2. Server Connectivity & Power BI Integration
Unlike static CSV projects, this dashboard is built on a live data foundation:
Server Link: Established a direct connection between MySQL and Power BI, ensuring the dashboard reflects real-time or scheduled data updates.I manually designed the relationship schema within Power BI to ensure efficient filtering and cross-highlighting between visuals. 


> Predicts which students are likely to complete online courses
> based on engagement behavior and demographics

- **Database:** MySQL
- **Records:** 500 students · 1,207 enrollments · 7,711 activity logs
- **Key techniques:** JOINs, CTEs, Views, Stored Procedures, 
                      Composite scoring, At-risk detection
- **Tools:** MySQL Workbench · Power BI
