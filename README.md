# SQL_Projects

A collection of end-to-end SQL projects built using MySQL, 
covering real-world scenarios in education..

---

## Projects

### 01 Student Course Completion Prediction
> Predicts which students are likely to complete online courses
> based on engagement behavior and demographics

- **Database:** MySQL
- **Records:** 500 students · 1,207 enrollments · 7,711 activity logs
- **Key techniques:** JOINs, CTEs, Views, Stored Procedures, 
                      Composite scoring, At-risk detection
- **Tools:** MySQL Workbench · Power BI
